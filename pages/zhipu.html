Latest Models and Products

GLM-5 LLM

GLM-5 (Feb 2026) is the bilingual reasoning foundation. GLM-5.1 (Mar 2026) is the coding flagship with strong agentic performance, trained on Huawei chips. Both released open-weight, with 5.1 following in April 2026.

GLM-4.7 LLM

Foundation model with improved coding, reasoning, and agentic capabilities and more reliable code generation.

GLM-5V-Turbo VLM

Vision-language coding model specialized for visual code generation, UI-to-code translation, and document understanding. Turbo variant optimized for low-latency inference.

AutoGLM AGENT

Agentic model and framework for autonomous task execution. Supports web browsing, form filling, app interaction, and multi-step workflow automation driven by natural language instructions.
