Latest Models and Products

Hermes 4 FINE-TUNE

Hermes 4 (70B/405B) on Llama 3.1 offers hybrid reasoning with strong tool-use and agent behavior. Hermes 4.3 (36B, Dec 2025) on ByteDance Seed delivers 70B-class performance with 512K context and the highest RefusalBench score of any model.

Hermes Agent AGENT

Open-source AI agent framework released Feb 2026. Features persistent memory, multi-platform support (14 messaging platforms), autonomous skill creation, and 6 execution backends. Surpassed 40,000 GitHub stars. MIT licensed.

DeepHermes 3 FINE-TUNE

Reasoning-focused fine-tune released February 2025. Extended chain-of-thought and step-by-step reasoning aligned for mathematical, logical, and scientific problem-solving. Builds on the Hermes fine-tuning methodology with explicit reasoning supervision.

Recent Activity and Context

NousResearch spans general fine-tunes (Hermes 4 family), reasoning specialists (DeepHermes 3), and agent infrastructure (Hermes Agent). The Hermes 4.3 model achieved the highest RefusalBench score globally. All models are open-weight.
