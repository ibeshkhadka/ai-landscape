Latest Models and Products

Claude Fable 5 LLM

Public Mythos-class frontier model released June 9, 2026. Top-tier reasoning, coding, and agentic performance. $10/$50 per MTok input/output. Represents the current frontier Claude generation.

Claude Opus 4.8 LLM

Advanced reasoning model released May 28, 2026 with 1M context window. Stronger across coding, agentic tasks, and multi-step tool orchestration. Dynamic Workflows feature enables orchestration of up to 1,000 parallel subagents.

Claude Sonnet 4.6 LLM

Balanced production model with near-Opus performance at lower cost. Strong tool use, computer use, and long-context reasoning for everyday Claude-powered products.

Claude Haiku 4.5 LLM

Fastest and most cost-effective Claude model. Optimized for high-throughput tasks, realtime chatbots, and lightweight agentic workflows. Ideal for latency-sensitive and budget-conscious production deployments.
